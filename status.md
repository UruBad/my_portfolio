### Реализованный функционал
- **Hero** — имя, роль, краткое «Обо мне», CTA: скачать резюме (PDF), написать, GitHub
- **Обо мне** — текст, локация, ожидания по доходу, формат занятости (RU/EN)
- **Опыт** — таймлайн по местам работы из резюме
- **Навыки** — категории и теги (языки, фреймворки, инструменты, верстка)
- **Проекты** — пустое состояние (структура готова к добавлению проектов)
- **Образование и курсы** — ВУЗ, курсы/сертификаты
- **Услуги** — карточки «Чем могу быть полезен»
- **Контакты** — телефон, email, GitHub, город (кликабельные ссылки)
- **i18n** — переключатель RU/EN, сохранение в `localStorage`
- **Тема** — светлая/тёмная, переключатель, сохранение в `localStorage`
- **Навигация** — плавный скролл по якорям, подсветка активной секции, бургер-меню на мобильных
- **Анимации** — появление секций при скролле, учёт `prefers-reduced-motion`
- **Доступность** — `:focus-visible`, aria-атрибуты, семантика

### Следующие шаги для тебя
1. В репозитории на GitHub: **Settings → Pages** → Source = **GitHub Actions**.
2. Пуш в ветку `main` запустит деплой; сайт будет по адресу `https://<username>.github.io/portfolio/`.
3. При обновлении резюме: `cp tasks/resume.pdf public/resume.pdf` и коммит.
